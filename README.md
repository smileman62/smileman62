안녕하세요! **UI/UX에 관심이 많은 프론트엔드 개발자 김진성**입니다.<br>
저는 제가 만든 화면이 **사용자와 소통하고**, 서비스가 전달하고자 하는 **가치를 자연스럽게 보여줄 때** 가장 큰 보람을 느낍니다.
